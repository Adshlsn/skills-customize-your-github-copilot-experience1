# 📘 Assignment: Data Structures in Python

## 🎯 Objective

Master the four fundamental Python data structures—lists, dictionaries, sets, and tuples—by implementing practical programs that demonstrate how to create, manipulate, and apply each structure effectively.

## 📝 Tasks

### 🛠️ Working with Lists

#### Description
Create a program that demonstrates list operations including creating lists, accessing elements, adding/removing items, and iterating through lists. Build a simple inventory management system where users can add products, remove products, and view the current inventory.

#### Requirements
Completed program should:

- Allow users to add items to a list with item names and quantities
- Allow users to remove items from the inventory
- Display the current inventory in a readable format
- Use list methods like append(), remove(), and sort()

### 🛠️ Building a Dictionary-Based Database

#### Description
Create a program that uses dictionaries to store and manage a simple database of students with their grades. Implement functionality to add students, update grades, calculate class statistics, and search for students.

#### Requirements
Completed program should:

- Store student names as keys and grades as values in dictionaries
- Allow adding new students and updating existing grades
- Calculate and display the class average, highest grade, and lowest grade
- Search for a student by name and return their grade
- Provide a function to display all students sorted by grade

### 🛠️ Using Sets for Unique Data

#### Description
Create a program that demonstrates set operations by analyzing student enrollment across multiple classes. Use sets to find overlapping enrollments, unique students, and generate enrollment reports.

#### Requirements
Completed program should:

- Define sets for each class with student names
- Find students enrolled in multiple classes (intersection)
- Find all unique students across all classes (union)
- Find students in only one class (difference operations)
- Demonstrate set operations with meaningful output

### 🛠️ Exploring Tuples for Immutable Data

#### Description
Create a program that uses tuples to store and manage immutable data like coordinates, RGB color values, or person information. Build a simple system that demonstrates when and why tuples are preferable to lists.

#### Requirements
Completed program should:

- Create tuples to represent compound data (e.g., coordinates as (x, y), colors as (r, g, b))
- Demonstrate tuple unpacking to extract individual values
- Use tuples as dictionary keys where appropriate
- Show why tuples are useful for protecting data from accidental modification
- Create functions that return multiple values as tuples
